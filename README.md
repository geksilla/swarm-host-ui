# swarm-host-ui

* `git clone <repository-url>` this repository
* `cd swarm-host-ui`
* `npm install`

## Running / Development

* Edit `app/adapters/application.js`. Change `host` and `namespace` to real values where `swarm-host` is running
* `ember serve`
* Visit your app at [http://localhost:4200](http://localhost:4200).
